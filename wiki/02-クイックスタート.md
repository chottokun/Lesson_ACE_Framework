# 🚀 2. クイックスタート

## 必要条件
- Python 3.12以上
- `uv` (高速なパッケージマネージャー)

## セットアップ

### 1. リポジトリのクローン
```bash
git clone <repository_url>
cd ace_rm
```

### 2. 依存関係のインストール
`uv` を使用してプロジェクトを同期します。
```bash
uv sync
```

### 3. 環境変数の設定
`.env.example` をコピーして `.env` ファイルを作成し、必要な情報を入力します。
```bash
cp .env.example .env
```

**主な設定項目 (`.env`):**
```env
# LLMの設定 (OpenAI互換API)
LLM_MODEL=gpt-oss-120b
LLM_API_KEY=your_api_key_here
LLM_BASE_URL=https://api.ai.sakura.ad.jp/v1/

# 言語設定 (ja: 日本語, en: 英語)
ACE_LANG=ja

# 記憶モード (shared: 共有, isolated: ユーザー別)
LTM_MODE=shared
```

## 実行方法

### Web UI (Gradio) の起動
エージェントとの対話や、内部状態（記憶、タスクキュー）を確認できるUIを起動します。
```bash
uv run python src/ace_rm/app.py
```
起動後、ブラウザで `http://127.0.0.1:7860` にアクセスしてください。

### 手動テストの実行
システムが正しく学習し、知識を再利用できるかを確認するためのスクリプトです。
```bash
uv run python tests/manual_test_memory_flow.py
```
